# Advent of Code 2020

My attempt at solving the series of programming puzzles from [Advent of Code](https://adventofcode.com/2020), in Go.

## Test

```sh
# for the latest day
make test

# for a particular day
make test day=01_report
```

## Build

```sh
# for the latest day
make build

# for a particular day
make build day=01_report
```

## Run

```sh
# for the latest day
make run

# for a particular day
make run day=01_report
```
